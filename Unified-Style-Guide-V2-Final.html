Unified Design System
for Calm, Clear Guides

One cohesive system merging three project design sources, optimized for ADHD-friendly, anxiety-conscious, and broadly accessible experiences.

13 color tokens + 6 state tokens Spacing scale Motion tokens Dark mode ready WCAG AA+
01

Warm, never clinical

Cream backgrounds. Soft ink. No pure white, no institutional navy.

02

Show one thing at a time

Progressive disclosure by default. Accordions, not walls of text.

03

Friendly type, always

Nunito for headings — rounded, approachable. Never formal serif as display.

04

Slow, smooth motion

250–400ms transitions. Nothing sudden or startling. Changes feel earned.

05

Errors feel safe

State colors inform, never alarm. Error red is calm, not crisis.

06

Gold is for wins only

Gold tokens are reserved for positive reinforcement. Never warnings.

Animation & transition
Motion Tokens
Slow is calm. Fast is startling. Every interactive state should feel like a breath, not a jolt.
/* Motion tokens — use these everywhere, never raw ms values */

--transition-quick:  150ms ease;          /* :active press, icon swap — near-instant */
--transition-base:   250ms ease;          /* hover, focus, color change — standard */
--transition-slow:   400ms ease;          /* accordion, progress fill — deliberate */
--transition-reveal: 350ms cubic-bezier(.16,1,.3,1); /* page load reveals, card entrance */

/* Usage examples */
.btn           { transition: filter var(--transition-base), transform var(--transition-quick); }
.accord-arrow  { transition: transform var(--transition-base); }
.progress-fill { transition: width var(--transition-slow); }
.field-input   { transition: border-color var(--transition-base), box-shadow var(--transition-base); }

/* Respect reduced motion preferences */
@media (prefers-reduced-motion: reduce) {
  * { transition-duration: 0.01ms !important; animation-duration: 0.01ms !important; }
}
