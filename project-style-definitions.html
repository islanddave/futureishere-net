CSS Custom Properties
Copy-paste ready · System 01 — Documentation Portal
/* System 01: Documentation Portal
   Aesthetic: Editorial dark-ink · Print-heritage · Warm paper
   Google Fonts: Playfair Display (400,600,800,italic) · DM Sans (300,400,500) · DM Mono (400,500) */

:root {
  /* ── Core ink scale ── */
  --ink:          #1a1814;   /* primary text, dark header bg */
  --ink-mid:      #3d3930;   /* secondary text */
  --ink-soft:     #7a7060;   /* muted / label copy */

  /* ── Paper surfaces ── */
  --paper:        #f5f0e8;   /* page / canvas background */
  --paper-warm:   #ede8dc;   /* card / panel background */
  --rule:         #d6cfc0;   /* dividers, borders */

  /* ── Rust accent (primary CTA) ── */
  --accent:       #c0622a;   /* links, buttons, active states */
  --accent-lt:    #e8a57a;   /* hover tint */
  --accent-bg:    #fdf3ec;   /* tinted surface / highlight band */

  /* ── Pine teal (secondary) ── */
  --teal:         #2a7a6a;   /* Guide 01 / secondary action */
  --teal-lt:      #a8d9cf;   /* teal badge bg / tint */
  --teal-bg:      #eef7f5;   /* section surface */

  /* ── Plum violet (tertiary) ── */
  --violet:       #5a3d8a;   /* Guide 02 / coming-soon tint */
  --violet-bg:    #f2eef9;   /* violet surface */

  /* ── Ochre gold (quaternary) ── */
  --gold:         #b08825;   /* tip callout stroke */
  --gold-bg:      #fdf6e3;   /* tip / note surface */

  /* ── Typography ── */
  --font-display: 'Playfair Display', Georgia, serif;
  --font-body:    'DM Sans', system-ui, sans-serif;
  --font-mono:    'DM Mono', 'Courier New', monospace;

  /* ── Gradient accent bar ── */
  --gradient-bar: linear-gradient(90deg, #c0622a 0%, #b08825 33%, #2a7a6a 66%, #5a3d8a 100%);
}

Platform Implementation Notes

Web

  • Use CSS custom properties as-is
  • Load fonts from Google Fonts CDN
  • Apply noise texture via SVG feTurbulence
  • Gradient bar: 3px top border on header
  • DM Mono for all eyebrow labels

iOS / Android

  • Bundle Playfair Display + DM Sans OTF
  • Map --accent to system tintColor
  • --paper → UIColor from hex #f5f0e8
  • DM Mono → monospaced system fallback
  • Skip noise texture (performance cost)

DOCX / PDF

  • Heading 1 = Playfair Display 800
  • Body = DM Sans 300 (or Calibri fallback)
  • Accent color on heading underlines
  • Teal/violet for callout box borders
  • Gold-bg for tip shading fills

CSS Custom Properties
Copy-paste ready · System 02 — ESL Helper Guide
/* System 02: ESL Helper Guide
   Aesthetic: Organic / natural · Warm, low-overwhelm · ADHD-friendly
   Google Fonts: Nunito (400,600,700,800,900) · Lora (400,600, italic 400) */

:root {
  /* ── Surfaces ── */
  --cream:              #fdf8f2;   /* page background */
  --warm-white:         #fffdf9;   /* card / modal surface */

  /* ── Sage (primary accent) ── */
  --sage:               #7a9e87;   /* headers, primary buttons */
  --sage-light:         #b8d4bf;   /* badge bg, tint */
  --sage-pale:          #e8f2eb;   /* section bg, hero gradient stop */

  /* ── Terracotta (secondary accent) ── */
  --terracotta:         #c8714a;   /* secondary CTA, icon color */
  --terracotta-light:   #f0c4ae;   /* terracotta tint */
  --terracotta-pale:    #fdf0e8;   /* warm notice / hero gradient stop */

  /* ── Lavender (tertiary accent) ── */
  --lavender:           #9b8ec4;   /* tip icon, badge, hero orb */
  --lavender-pale:      #ede9f7;   /* lavender surface */

  /* ── Golden (highlight / reward) ── */
  --golden:             #d4a843;   /* star, copy-button, celebration */
  --golden-pale:        #fdf3dc;   /* tip / note surface */

  /* ── Text ── */
  --text-dark:          #2d2d2d;   /* primary body text */
  --text-mid:           #5a5a5a;   /* secondary text */
  --text-soft:          #888888;   /* muted / helper text */
  --border-soft:        rgba(0,0,0,.07); /* card borders */

  /* ── Typography ── */
  --font-display:  'Nunito', 'Helvetica Rounded', sans-serif;
  --font-body:     'Lora', Georgia, serif;

  /* ── Hero gradient (for backgrounds) ── */
  --hero-gradient: linear-gradient(145deg, #e8f2eb 0%, #fdf3dc 50%, #fdf0e8 100%);
}

Platform Implementation Notes

Web

  • Hero uses 145deg gradient of all three pale stops
  • Accordion cards: white bg, border-soft shadow
  • Copy buttons: --golden bg, white text, rounded pill
  • Smooth max-height accordion animation (0.3s ease)
  • Nunito 900 for emotional / big moments

iOS / Android

  • Bundle Nunito + Lora OTF files
  • --cream as tableView.backgroundColor
  • Collapsible sections → UITableView groups
  • --sage as primary tintColor system-wide
  • --golden for positive feedback / success states

DOCX / PDF

  • Nunito 900 = Title style
  • Nunito 700 = Heading 1, Heading 2
  • Lora 400 = Normal (body)
  • Sage-pale as shading for info boxes
  • Terracotta for step-number circles

CSS Custom Properties
Copy-paste ready · System 03 — Organizing & Productivity Guide
/* System 03: Organizing & Productivity Guide
   Aesthetic: Corporate professional · Navy blue primary · High-contrast
   Fonts: System UI / Calibri (no web font dependency — works offline)
   Note: originated as a DOCX; HTML layer maps Word styles to CSS tokens */

:root {
  /* ── Blue scale (primary brand) ── */
  --blue-deep:      #1F4E79;   /* H1, H2, dark header bg, table header text */
  --blue-mid:       #2E75B6;   /* H3, links, active tabs, step numbers */
  --blue-light:     #D6E4F0;   /* table header bg, tinted callout surface */

  /* ── Orange accent ── */
  --accent-orange:  #C55A11;   /* tip callout border, step accent, icon color */

  /* ── Surfaces ── */
  --surface-white:  #ffffff;   /* card, page, modal bg */
  --surface-off:    #f8f9fa;   /* alternate row, section bg */
  --border:         #dee2e6;   /* dividers, card borders, table lines */

  /* ── Text ── */
  --text-primary:   #212529;   /* body text */
  --text-muted:     #6c757d;   /* secondary / label text */

  /* ── Typography ── */
  --font-display:   system-ui, 'Segoe UI', Calibri, sans-serif;
  --font-body:      system-ui, 'Segoe UI', Calibri, sans-serif;
  /* DOCX equivalents: Heading 1–2 = Calibri Bold #1F4E79 */
  /*                   Heading 3    = Calibri Bold #2E75B6 */
  /*                   Normal       = Calibri 11pt #212529 */

  /* ── Step component (numbered steps) ── */
  --step-num-bg:    #2E75B6;   /* step circle background */
  --step-title-color:#1F4E79;  /* step title text */
  --step-desc-color: #6c757d;  /* step description text */
}

Platform Implementation Notes

Web

  • System font stack — no Google Fonts needed
  • Step component: circle badge + title + desc
  • Tables: --blue-light th, --border td rules
  • Orange accent for instructional tip callouts
  • Works offline / no CDN dependency

iOS / Android

  • System font fallbacks work natively
  • --blue-deep → navigationBar.barTintColor
  • --blue-mid → tintColor (links/buttons)
  • --accent-orange → warning / tip badge
  • Steps → numbered UITableView rows

DOCX / PDF

  • Direct Word style mapping (see table above)
  • Calibri is the native font — no embedding needed
  • Table headers: --blue-light fill + --blue-deep text
  • Tip boxes: left-border #C55A11, pale bg fill
  • PDF: all styles preserve via Word export
